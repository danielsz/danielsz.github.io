The Clojurescript quick start guide has been converted into a Homebrew formula, but since that move has proven to be contentious, and since I have no idea how long it will take before it gets approved, if ever, I made the formula available directly from a github repository.

All you need to do is: 

brew tap danielsz/clojurescript
brew install clojurescript

This will put the `clojurescript` compiler as an independent executable on your path, but please note that for most intents and purposes, you will want to work with `leiningen` and add `clojurescript` as a project dependency.
